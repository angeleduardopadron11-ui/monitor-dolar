#  Venezuela Exchange Rate Monitor (BCV)
A Python-based web scraper that fetches the official exchange rate from the Central Bank of Venezuela (BCV).

##  Tech Stack
- **Language**: Python 3.12
- **Environment**: KDE Neon (Linux)
- **Libraries**: BeautifulSoup4, Requests

##  How to use
1. Activate virtual environment: `source venv/bin/activate`
2. Install dependencies: `pip install -r requirements.txt`
3. Run: `python3 monitor.py`
